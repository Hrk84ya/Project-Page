## üîß Setup & Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/Hrk84ya/Project-Page.git
   cd Project-Page
   ```

2. **Serve locally**
   ```bash
   # Using Python
   python3 -m http.server 8000

   ```

